# Entendendo a Estrutura Digital - PharmaCorp

## O que é este projeto?
Este diagrama representa a "planta baixa" de como as informações e sistemas da PharmaCorp funcionam dentro da internet (nuvem). O objetivo não é apenas ter um sistema rápido, mas garantir que as fórmulas e pesquisas da empresa estejam tão seguras quanto estariam em um cofre físico.
